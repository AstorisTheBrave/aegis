# Code of Conduct

I want Aegis to be a constructive place to build. This code applies to issues,
pull requests, discussions, releases, and every other project space.

Be respectful, assume good faith, welcome different experience levels, and keep
feedback about the work. Harassment, discrimination, threats, doxxing, and
personal attacks are not acceptable.

If something crosses that line, report it privately through the repository's
[security advisory channel](https://github.com/AstorisTheBrave/aegis/security/advisories/new).
Maintainers will review reports promptly and may remove content or participation
when needed to protect the community.
